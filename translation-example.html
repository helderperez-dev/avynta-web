How to Use

1. <t key="translation.key">Default Text</t>

Use custom <t> tags with a key attribute. The default text will be replaced with the translation.

2. <span data-t="translation.key">Default Text</span>

Use data-t attribute on any element. Works with any HTML tag.

3. <input data-t="translation.key" placeholder="Default">

For form elements, the translation will be used as placeholder text.

4. <button data-t="translation.key">Default</button>

Button text will be replaced with the translation.
